enum Error_Codes_CFS

Error codes for the calculate_file_size() function.

CFS - Calculate File Size. These error codes help to identify various issues that can occur while determining the size of a file.

Note
Each error code corresponds to a specific failure during file handling or file size calculation.
See also
calculate_file_size() for function utilizing these error codes.
Return values
CFS_SUCCESSThe operation completed successfully, no errors occurred.
CFS_NULL_FILE_PATH_POINTERThe file path argument was NULL.
CFS_NULL_FILE_SIZE_POINTERThe file size pointer argument was NULL.
CFS_FILE_FOPEN_ERRORThe file could not be opened.
CFS_END_FSEEK_ERRORFailed to set the file position to the end of the file.
CFS_SIZE_FTELL_ERRORFailed to retrieve the current value of the file-position indicator.
CFS_FILE_FCLOSE_ERRORThe file could not be closed properly.
