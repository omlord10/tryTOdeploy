enum Error_Codes_RS

Error codes for the ReadSignature function.

These codes represent different failure scenarios that might occur when reading a virus signature from a file.

Note
Each error code corresponds to a specific issue during file handling or signature parsing. Use these error codes to troubleshoot issues with file reading operations in the ReadSignature function.
See also
ReadSignature() for function utilizing these error codes.
Return values
RS_SUCCESSNo errors, the operation completed successfully.
RS_NULL_FILE_PATH_POINTERThe file path argument was NULL.
RS_NULL_VSTRUCT_POINTERThe virus structure pointer was NULL.
RS_FILE_FOPEN_ERRORThe file could not be opened.
RS_SIGNATURE_FSCANF_ERRORSignature reading from the file failed.
RS_OFFSET_FSCANF_ERRORSetting the file position failed.
RS_VNAME_FSCANF_ERRORReading the virus name from the file failed.
RS_FILE_FCLOSE_ERRORThe file could not be closed properly.
